Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
0
.
.
9
deadlock in 14 line

for-range цикл читает с канала до тех пор пока он не закрыт,
канал никогда не закрывается, нет горутин пишущих в канал - происходит дедлок 
```
